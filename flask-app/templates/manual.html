User manual for math prompt

Here are some commands to write in fields with math prompt enabled

  • ∫ - type "int"
  • ∑ - type "sum"
  • √ - type "sqrt"
  • π - type "pi"
  • ∞ - type "infty" or "infinity"
  • Type "^" for making sup
  • Type "_" for making sub
  • Type "\text" + space for writing text in area fields with math prompt

If you have questions about more complicated structures, you can type in LaTeX
